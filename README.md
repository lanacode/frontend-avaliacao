# frontend-avaliacao
```
simulador-apostas/
│
├── index.html → página principal (menu de esportes)
├── futebol.html → seção de apostas em futebol
├── basquete.html → seção de apostas em basquete
├── tenis.html → seção de apostas em tênis
├── formula1.html → seção de apostas em Fórmula 1
├── esport.html → seção de apostas em e-sports
│
├── login.html → tela de login e cadastro de usuário
├── perfil.html → carteira, saldo e histórico de apostas
│
├── /css
│ ├── style.css → estilos gerais
│ ├── esportes.css → estilos dos cards e tabelas de jogos
│ ├── perfil.css → layout da carteira e histórico
│ ├── login.css → tela de login/cadastro
│ ├── responsivo.css → ajustes mobile/tablet
│
├── /js
│ ├── main.js → scripts globais (menu, navegação, dark mode)
│ ├── dados.js → JSON fictício com jogos e odds
│ ├── apostas.js → lógica de apostas e simulação de ganhos
│ ├── usuario.js → login, cadastro, saldo e manipulação de dados
│ ├── carteira.js → depósito, saque e histórico do usuário
│
└── /img → logos, ícones, esportes, avatares e demais assets
```
